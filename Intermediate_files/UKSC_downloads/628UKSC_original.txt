<span style="letter-spacing:.95pt"> </span>care who is looking after him is that medication can be used to ensure that he is not <span style="letter-spacing: 2.5pt"> </span>aware of the NIV being withdrawn and does not become uncomfortable and <span style="letter-spacing:-.25pt"> </span>distressed. </span></a></p><a name="para3">
 <p class="MsoNormal" style="margin-top:.05pt"> <span lang="EN-US" style="font-size: 12.0pt;font-family:&quot;Times New Roman&quot;,&quot;serif&quot;">&nbsp; </span></p>
 </a><p class="MsoBodyText" style="margin-right:5.85pt;text-align:justify"><a name="para3"> <span lang="EN-US"></span></a><a name="para4">4. <span style="font:7.0pt &quot;Times New Roman&quot;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>  <span lang="EN-US">However, <span style="letter-spacing:1.5pt"> </span>Mr <span style="letter-spacing:1.6pt"> </span>Conway <span style="letter-spacing:1.3pt"> </span>does <span style="letter-spacing:1.55pt"> </span>not <span style="letter-spacing:1.55pt"> </span>accept <span style="letter-spacing:1.55pt"> </span>that <span style="letter-spacing:1.5pt"> </span>the <span style="letter-spacing:1.75pt"> </span>withdrawal <span style="letter-spacing: 1.55pt"> </span>of <span style="letter-spacing:1.5pt"> </span>his <span style="letter-spacing:1.55pt"> </span>NIV <span style="letter-spacing:1.6pt"> </span>under <span style="letter-spacing:1.5pt"> </span>heavy sedation would be a dignified death. He does not know how he would feel, whether <span style="letter-spacing:1.65pt"> </span>he would <span style="letter-spacing:1.95pt"> </span>experience <span style="letter-spacing:1.9pt"> </span>the <span style="letter-spacing:2.0pt"> </span>drowning <span style="letter-spacing:1.85pt"> </span>sensation <span style="letter-spacing: 1.95pt"> </span>of <span style="letter-spacing:1.9pt"> </span>not <span style="letter-spacing:2.1pt"> </span>being <span style="letter-spacing:2.0pt"> </span>able <span style="letter-spacing:1.95pt"> </span>to <span style="letter-spacing:2.0pt"> </span>breathe, <span style="letter-spacing:2.05pt"> </span>whether <span style="letter-spacing:1.9pt"> </span>he would <span style="letter-spacing:1.15pt"> </span>be <span style="letter-spacing:1.1pt"> </span>able <span style="letter-spacing:1.1pt"> </span>to <span style="letter-spacing:1.2pt"> </span>hear <span style="letter-spacing:1.15pt"> </span>his <span style="letter-spacing:1.2pt"> </span>family <span style="letter-spacing:1.15pt"> </span>Taking <span style="letter-spacing:1.05pt"> </span>lethal <span style="letter-spacing:1.2pt"> </span>medicine <span style="letter-spacing:1.1pt"> </span>would avoid <span style="letter-spacing: 1.25pt"> </span>all <span style="letter-spacing:1.25pt"> </span>these <span style="letter-spacing:1.15pt"> </span>problems. <span style="letter-spacing: 1.4pt"> </span> <span style="letter-spacing:-.15pt">In </span> <span style="letter-spacing:1.25pt"> </span>his <span style="letter-spacing:1.25pt"> </span>view, <span style="letter-spacing:1.25pt"> </span>which <span style="letter-spacing:1.25pt"> </span>is <span style="letter-spacing:1.25pt"> </span>shared <span style="letter-spacing:1.25pt"> </span>by <span style="letter-spacing:.85pt"> </span>many, <span style="letter-spacing:1.25pt"> </span>it <span style="letter-spacing:1.25pt"> </span>is <span style="letter-spacing:1.25pt"> </span>his <span style="letter-spacing:1.15pt"> </span>life <span style="letter-spacing:1.15pt"> </span>and <span style="letter-spacing:1.25pt"> </span>he should have the right to choose to end it in the way which he considers most <span style="letter-spacing:-.25pt"> </span>consistent with his human <span style="letter-spacing:-.05pt"> </span>dignity. </span></a></p><a name="para4">
 <p class="MsoNormal"> <span lang="EN-US" style="font-size:12.0pt;font-family:&quot;Times New Roman&quot;,&quot;serif&quot;">&nbsp; </span></p>
 </a><p class="MsoBodyText" style="margin-right:5.55pt;text-align:justify"><a name="para4"> <span lang="EN-US"></span></a><a name="para5">5. <span style="font:7.0pt &quot;Times New Roman&quot;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>  <span lang="EN-US">The question for this panel is whether his case raises an arguable point of law <span style="letter-spacing:1.6pt"> </span>of general public importance which ought to be heard by the Supreme Court at this <span style="letter-spacing:1.65pt"> </span>time. No-one <span style="letter-spacing: 1.25pt"> </span>doubts <span style="letter-spacing:1.35pt"> </span>that <span style="letter-spacing:1.3pt"> </span>the <span style="letter-spacing:1.3pt"> </span>issue <span style="letter-spacing:1.25pt"> </span>is <span style="letter-spacing:1.35pt"> </span>of <span style="letter-spacing:1.25pt"> </span>transcendent <span style="letter-spacing: 1.35pt"> </span>public <span style="letter-spacing:1.25pt"> </span>importance. <span style="letter-spacing:1.4pt"> </span>It <span style="letter-spacing:1.35pt"> </span>touches <span style="letter-spacing:1.3pt"> </span>us <span style="letter-spacing:1.3pt"> </span>all. We <span style="letter-spacing:1.05pt"> </span>all <span style="letter-spacing: 1.15pt"> </span>have <span style="letter-spacing:1.15pt"> </span>to <span style="letter-spacing:1.15pt"> </span>experience <span style="letter-spacing: 1.05pt"> </span>the <span style="letter-spacing:1.2pt"> </span>death <span style="letter-spacing:1.15pt"> </span>of <span style="letter-spacing:1.2pt"> </span>people <span style="letter-spacing:1.2pt"> </span>about <span style="letter-spacing:1.15pt"> </span>whom <span style="letter-spacing:1.1pt"> </span>we <span style="letter-spacing:1.15pt"> </span>care. <span style="letter-spacing:1.2pt"> </span>We <span style="letter-spacing:1.05pt"> </span>all <span style="letter-spacing:1.15pt"> </span>have <span style="letter-spacing:1.05pt"> </span>to contemplate our own <span style="letter-spacing:-.1pt"> </span>death. </span></a></p><a name="para5">
 <p class="MsoNormal"> <span lang="EN-US" style="font-size:12.0pt;font-family:&quot;Times New Roman&quot;,&quot;serif&quot;">&nbsp; </span></p>
 </a><p class="MsoBodyText" style="margin-top:5.1pt;margin-right:5.55pt;margin-bottom: 0cm;margin-left:23.9pt;margin-bottom:.0001pt;text-align:justify"><a name="para5"> <span lang="EN-US"></span></a><a name="para6">6. <span style="font:7.0pt &quot;Times New Roman&quot;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>  <span lang="EN-US">&nbsp;No-one <span style="letter-spacing:1.65pt"> </span>doubts <span style="letter-spacing:1.75pt"> </span>that <span style="letter-spacing:1.7pt"> </span>the <span style="letter-spacing:1.7pt"> </span>issue <span style="letter-spacing:1.65pt"> </span>raises <span style="letter-spacing:1.7pt"> </span>arguable <span style="letter-spacing:1.65pt"> </span>points <span style="letter-spacing:1.75pt"> </span>of <span style="letter-spacing:1.7pt"> </span>law. <span style="letter-spacing:1.7pt"> </span>The <span style="letter-spacing:1.65pt"> </span>difference <span style="letter-spacing: 1.65pt"> </span>between letting someone die and taking active steps to bring about their death has been <span style="letter-spacing:2.8pt"> </span>central to the common law for centuries. Some argue that to depart from that distinction is <span style="letter-spacing:1.45pt"> </span>to cross a dangerous Rubicon. Some argue that the distinction is morally and <span style="letter-spacing:1.4pt"> </span>practically indefensible. <span style="letter-spacing:.8pt"> </span>The <span style="letter-spacing:.75pt"> </span>Human <span style="letter-spacing:.8pt"> </span>Rights <span style="letter-spacing:.85pt"> </span>Act <span style="letter-spacing:.85pt"> </span>1998 <span style="letter-spacing:.8pt"> </span>has <span style="letter-spacing:.85pt"> </span>added <span style="letter-spacing:.8pt"> </span>a <span style="letter-spacing:.8pt"> </span>new <span style="letter-spacing:.8pt"> </span>dimension <span style="letter-spacing:.85pt"> </span>to <span style="letter-spacing:.8pt"> </span>the <span style="letter-spacing: .8pt"> </span>debate. All <span style="letter-spacing:1.85pt"> </span>are <span style="letter-spacing:1.75pt"> </span>agreed <span style="letter-spacing:1.85pt"> </span>that <span style="letter-spacing:1.85pt"> </span>the <span style="letter-spacing:1.95pt"> </span>ban <span style="letter-spacing:1.85pt"> </span>on <span style="letter-spacing:1.85pt"> </span>assisting <span style="letter-spacing: 1.7pt"> </span>suicide <span style="letter-spacing:1.9pt"> </span>is <span style="letter-spacing:1.85pt"> </span>an <span style="letter-spacing:1.85pt"> </span>interference <span style="letter-spacing:1.8pt"> </span>with <span style="letter-spacing:2.0pt"> </span>the <span style="letter-spacing:1.8pt"> </span>right <span style="letter-spacing:1.85pt"> </span>to respect for private life protected by article 8. As the European Court of Human <span style="letter-spacing:.8pt"> </span>Rights has <span style="letter-spacing:.75pt"> </span>said, <span style="letter-spacing:.8pt"> </span>in <span style="letter-spacing: .85pt"> </span> </span> <i> <span lang="EN-US">Pretty <span style="letter-spacing: .7pt"> </span>v <span style="letter-spacing:.7pt"> </span>United <span style="letter-spacing:.75pt"> </span>Kingdom <span style="letter-spacing:.85pt"> </span> </span> </i> <span lang="EN-US"> </span></a><a title="Link to BAILII version" href="/cgi-bin/redirect.cgi?path=/eu/cases/ECHR/2002/427.html">(2002) 35 EHRR 1</a>, <span style="letter-spacing:.75pt"> </span>para <span style="letter-spacing:.65pt"> </span>65, <span style="letter-spacing:.75pt"> </span>‘The <span style="letter-spacing:.7pt"> </span>very <span style="letter-spacing:.65pt"> </span>essence of the Convention is respect for human dignity and human freedom’. But <span style="letter-spacing:2.1pt"> </span>several  <span lang="EN-US">justifications have been put forward to support a hard and fast rule </span> <span lang="EN-US">– </span> <span lang="EN-US">the protection <span style="letter-spacing:.45pt"> </span>of weak <span style="letter-spacing:1.6pt"> </span>and <span style="letter-spacing:1.6pt"> </span>vulnerable <span <span style="letter-spacing: 1.6pt"> </span>respect <span style="letter-spacing:1.6pt"> </span>for <span style="letter-spacing:1.5pt"> </span>the <span style="letter-spacing:1.55pt"> </span>sanctity <span style="letter-spacing:1.0pt"> </span>in <span style="letter-spacing:1.15pt"> </span> </span> <i> <span lang="EN-US">Nicklinson <span style="letter-spacing:1.0pt"> </span>v <span style="letter-spacing:.95pt"> </span>United <span style="letter-spacing:.95pt"> </span>Kingdom <span style="letter-spacing:1.15pt"> </span> </span> </i> <span lang="EN-US"><a title="Link to BAILII version" href="/cgi-bin/redirect.cgi?path=/eu/cases/ECHR/2015/709.html">(2015) 61 EHRR SE7</a>, <span style="letter-spacing:1.0pt"> </span>that whether <span style="letter-spacing: 1.3pt"> </span>an <span style="letter-spacing:1.25pt"> </span>interference <span style="letter-spacing:1.3pt"> </span>such <span style="letter-spacing:1.25pt"> </span>as <span style="letter-spacing:1.35pt"> </span>this <span style="letter-spacing:1.25pt"> </span>is <span style="letter-spacing:1.25pt"> </span>justified <span style="letter-spacing: 1.25pt"> </span>is <span style="letter-spacing:1.25pt"> </span>for <span style="letter-spacing:1.25pt"> </span>each <span style="letter-spacing:1.25pt"> </span>Member <span style="letter-spacing:1.2pt"> </span>State <span style="letter-spacing:1.3pt"> </span>to <span There is no European consensus on the <span style="letter-spacing:-.2pt"> </span>matter. </span></p>
 <p class="MsoNormal"> <span lang="EN-US" style="font-size:12.0pt;font-family:&quot;Times New Roman&quot;,&quot;serif&quot;">&nbsp; </span></p>
 <p class="MsoBodyText" style="margin-right:5.75pt;text-align:justify"> <span lang="EN-US"><a name="para7">7. <span style="font:7.0pt &quot;Times New Roman&quot;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span> </a></span><a name="para7"> <span lang="EN-US">Under the United Kingdom’s constitutional arrangements, only Parliament <span style="letter-spacing:1.4pt"> </span>could </span> <span lang="EN-US">change <span style="letter-spacing:.65pt"> </span>this <span style="letter-spacing:.7pt"> </span>law. <span style="letter-spacing:.7pt"> </span>But <span style="letter-spacing:.75pt"> </span>the <span style="letter-spacing:.8pt"> </span>Supreme <span style="letter-spacing:.7pt"> </span>Court <span style="letter-spacing:.7pt"> </span>could, <span style="letter-spacing:.75pt"> </span>if <span style="letter-spacing:.55pt"> </span>it <span style="letter-spacing:.75pt"> </span>thought <span style="letter-spacing:.75pt"> </span>right, <span style="letter-spacing:.75pt"> </span>make <span style="letter-spacing:.65pt"> </span>a <span style="letter-spacing:.8pt"> </span>declaration that <span style="letter-spacing:.85pt"> </span>the <span style="letter-spacing: .85pt"> </span>law <span style="letter-spacing:.85pt"> </span>was <span style="letter-spacing:.9pt"> </span>incompatible <span style="letter-spacing: .85pt"> </span>with <span style="letter-spacing:.9pt"> </span>the <span style="letter-spacing:.85pt"> </span>Convention <span style="letter-spacing: .85pt"> </span>rights, <span style="letter-spacing:.9pt"> </span>leaving <span style="letter-spacing:.75pt"> </span>it <span style="letter-spacing:.9pt"> </span>to <span style="letter-spacing:.9pt"> </span>Parliament <span style="letter-spacing:.85pt"> </span>to decide what, if anything, to do about it.  <p class="MsoNormal"> <span lang="EN-US" style="font-size:12.0pt;font-family:&quot;Times New Roman&quot;,&quot;serif&quot;">&nbsp; </span></p>
 </a><p class="MsoBodyText" style="margin-right:5.75pt;text-align:justify"><a name="para7"> <span lang="EN-US"></span></a><a name="para8">8. <span style="font:7.0pt &quot;Times New Roman&quot;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>  <span lang="EN-US">Ultimately, the question for the panel is whether the </span> <span lang="EN-US">prospects of Mr <span style="letter-spacing: .55pt"> </span>Conway’s </span> <span lang="EN-US">succeeding in his claim before this court are sufficient to justify our giving <span style="letter-spacing: .3pt"> </span>him permission to pursue it, with all that that would entail for him, for his family, for <span style="letter-spacing:1.35pt"> </span>those on <span style="letter-spacing:.65pt"> </span>all <span style="letter-spacing:.65pt"> </span>sides <span style="letter-spacing:.65pt"> </span>of <span style="letter-spacing:.6pt"> </span>this <span style="letter-spacing:.65pt"> </span>multi-faceted <span style="letter-spacing: .6pt"> </span>debate, <span style="letter-spacing:.6pt"> </span>for <span style="letter-spacing:.55pt"> </span>the <span style="letter-spacing:.75pt"> </span>general <span style="letter-spacing:.65pt"> </span>public <span style="letter-spacing:.6pt"> </span>and <span style="letter-spacing:.75pt"> </span>for <span style="letter-spacing:.55pt"> </span>this <span style="letter-spacing:.65pt"> </span>court. <span style="letter-spacing:.6pt"> </span>Not without some reluctance, it has been concluded that in this case those prospects are <span style="letter-spacing:-.2pt"> </span>not sufficient to justify giving permission to <span style="letter-spacing:-.35pt"> </span>appeal. </span></a></p><a name="para8"> </a></div><a name="para8">
